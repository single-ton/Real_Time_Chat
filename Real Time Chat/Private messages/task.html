Description

In this stage, implement sending and receiving private messages. When, for example, user A sends a message to user B only these two users should see the message. Furthermore, after a user clicks on a username in the user list, the message container should be refilled with messages if these users have already chatted.

Theory

This Realtime Chatroom application - SpringBoot, Websocket, ReactJS video by InvolveInInnovation shows how to send a message to a specific user.

Objectives
  • Allow users to send private messages. The send message button should work differently, depending on which chat is opened. If a public chat is opened, the button should send a public message. If a private chat is opened between two users, after a message is sent only these two users should receive the message;
  • After the current user opens a private chat, fill the message container with the messages if the users have already chatted.
